# 当前路线图

## 当前阶段

当前阶段可以定义为：MVP 已跑通，正在补齐长期可维护性与可迭代性。

## 当前优先级

### P0

- 稳定素材导入、异步分析、检索、生成主链路
- 保持 `dev` 启动链路稳定
- 让新 AI 会话可通过 `docs/` 快速接手项目

### P1

- 提升点评与精选候选分析的解释性
- 提升检索质量与向量索引策略
- 补充更多 controller / integration 级测试
- 补充部署和排障文档

### P2

- 人工复评与覆盖 AI 结论
- 多版本生成与改写工作台
- 更完整的前端工作台能力

## 当前明确不做

- 权限系统
- 多租户
- 自动全网采集
- 完整发布闭环
- 复杂审批流

## 当前技术债

- PostgreSQL 高维向量索引策略仍需优化
- 测试环境与真实 PostgreSQL 之间仍存在能力差距
- 静态前端工作台仍可继续产品化
